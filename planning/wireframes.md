## Wireframe 1: Home (Interactive Map View)

![Home - Interactive Map View](https://github.com/user-attachments/assets/1f0ca1b4-325d-4abc-9353-b25c75cc79f9)

**Description:** The home page features an interactive map displaying yard sales as pins. The map includes:
- Legend showing "Happening now" (green) and "Upcoming" (teal) sales
- Location set to Orlando, FL
- Search bar for finding sales
- "Saved (2)" section for bookmarked sales
- "Host a Sale" button
- Filter option
- List view showing sales with details (address, distance, time, items, attendees)
- "Check in!" feature to see who's going

## Wireframe 2: Yard Sale Detail View

![Yard Sale Detail](https://github.com/user-attachments/assets/5ce14cfd-21ab-48a2-8dee-b2a0ccb43744)

**Description:** Detailed view of a specific yard sale showing:
- Sale status (HAPPENING NOW)
- Address and distance (543 Oak Street, 0.3 mi)
- Time (Today 8am - 1pm)
- Sale description: "Moving sale! High quality furniture, vintage books, gaming consoles and more. Everything must go!"
- Host information (Lin Family, 4.7 rating) with "Message Seller" button
- "Who's going?" section with attendee avatars
- "Check in!" button
- Items for sale grid with photos, prices, conditions (e.g., Mid century dresser $75 - Excellent, PlayStation 4 Bundle $100 - Good)
- "Save Item" buttons for each item
- Action buttons: "Get Directions", "Share", "Saved"
- Location map
